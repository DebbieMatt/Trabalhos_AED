## 🗂️ Detalhes dos Trabalhos

### 📁 [AED I — Caixa de Supermercado](https://github.com/DebbieMatt/Trabalhos_AED/tree/0cb8b3c8d5ef9a8a64164885c1a583e084e3a680/TRABALHO%20AED%201%20-%20Supermercado)

Simulação das atividades de um caixa e crediário de supermercado. O programa é capaz de:

- Implementar um menu de operações para o caixa
- Simular compras lendo código de produto e quantidade
- Aceitar múltiplas formas de pagamento, incluindo crediário
- Realizar controle de estoque
- Consultar e pagar saldo devedor de clientes
- Sortear clientes para perdão de dívida
- Aplicar descontos para compras no atacado

---

### 📁 [AED II — Análise de Algoritmos de Ordenação](https://github.com/DebbieMatt/Trabalhos_AED/tree/0cb8b3c8d5ef9a8a64164885c1a583e084e3a680/TRABALHO%20AED%202%20-%20An%C3%A1lise%20de%20Algoritmos%20de%20Ordena%C3%A7%C3%A3o)

Trabalho avaliativo focado na análise e comparação de desempenho de diferentes algoritmos de ordenação, estudando complexidade e eficiência em distintos cenários.

---

### 📁 [AED II — Clínica Médica](https://github.com/DebbieMatt/Trabalhos_AED/tree/0cb8b3c8d5ef9a8a64164885c1a583e084e3a680/TRABALHO%20AED%202%20-%20Clinica%20Medica)

Sistema de gerenciamento para uma clínica médica, aplicando estruturas de dados para organização e controle de informações de pacientes e atendimentos.

---

## 💻 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:

- Versão mais recente do `GCC` instalada, **ou** utilize um compilador online como o [OnlineGDB](https://www.onlinegdb.com/online_c++_compiler)
- Sistema operacional: `Windows`, `Linux` ou `macOS`
